# Agent Context Handoff

This folder is generated by Layer 2 and consumed by Layer 3 and Layer 4.

- layer1-requirements.json: Jira requirements extracted by Layer 1.
- layer2-strategy-context.json: strategy, MCP verification, retrieval queries, and codegen hints.
- layer2-test-plan.json: normalized test cases for code generation.
- playwright-verification.json: current Playwright listing/run result used to compare strategy coverage.
- explorer-context.json: browser-discovered a11y snapshot, UI elements, selectors, and flow coverage.
- layer3-retrieval-context.json: reusable assets discovered by Layer 3.

Layer 2 uses non-destructive Playwright listing by default. Actual test execution is opt-in:

```bash
node agents/agent-layer2-strategy.js ISE-1556 --run-playwright
```

Explorer browser discovery is also opt-in:

```bash
node agents/agent-layer2-strategy.js ISE-1556 --explore
```
